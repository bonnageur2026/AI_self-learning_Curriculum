# AI/ML Curriculum for a Mathematically Advanced Reader

This repository contains a self-designed curriculum for studying machine learning and artificial intelligence from a mathematically serious perspective.

The curriculum is organized around textbooks as the main foundation, with landmark papers used to trace the historical and conceptual development of the field.

## Topics

- core machine learning
- deep learning
- generative models
- transformers and large language models
- reinforcement learning
- causality
- mathematical bridges to research

## Purpose

This curriculum is intended for readers who want:

- strong theoretical foundations
- a structured path through modern AI/ML
- a reading program oriented toward research rather than short-term trend-following

It is not primarily a job-training or engineering roadmap.

## Audience

- mathematically advanced self-learners
- students preparing for research
- mathematicians entering AI/ML
- readers who prefer depth over breadth

## Use

Recommended approach:

1. follow the sequence
2. study textbooks first
3. use papers as landmarks
4. go slowly and deeply
5. add implementation exercises separately if needed
